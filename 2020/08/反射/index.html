反射（Reflect）

反射关键对象：class文件对象

类加载器创建，获取使用

三种方法：

​ 1：对象的方法getClass()获取（getClass()定义在Object类）

​ 2: 使用类的静态属性class获取

​ 3：Class类静态方法forName()获取（常用：扩展性好）

暴力反射方法执行方法：Class类类名.getDeclaredConstructor()

不建议使用

注意:方法后面加s则去除所有构造

Constructor 构造方法对象

Filed 成员变量对象(字段)

Method 成员方法对象

三者共同继承了父类AccessibleObject

父类方法setAccessible(true) 取消Java运行时期的安全检查

在暴力反射后使用

随笔

properties文件建立在src文件下

项目开发完毕，交给用户使用，是编译后的class文件

src源代码，编译后产生class文件，是同步的

如何读取src文件，使用类的加载器

ClassLoader 类定义方法

​ InputStream getResourceAsStream(“文件名”) 返回字节输入流

​ 此流会自动从类目录下扫描文件读取

public class Test{
    //Test的类加载器，必须先获取到这个类的class文件对象
    ClassLoader loader=Test.class.getClassLoader();
    //System.out.println(loader);//Launcher$AppClassLoader@18b4aac2
    //加载器获取字节输入流
    InputStream inputStream=loader.getResourceAsStream("config.properties");
    System.out.println(inputStream);
    //流对象和集合关联使用
    Properties properties=new Properties();
    properties.load(inputStream);
    inputStream.close();
    System.out.println(properties);
    //集合方法，获取值
    String className=properties.getProperty("className");
    String methodName=properties.getProperty("methodName");
    //反射，获取这个类的class文件对象
    Class c=Class.forName(className);
    //创建对象
    Object object=c.newInstance();
    //获取方法
    Method method=c.getMethod(methodName);
    //运行方法
    method.invoke(object);
