我的第一篇博客——博客相关命令

1.创建博客

1.hugo new posts/FileName.md

2.hugo server -t pure –buildDrafts //测试静态博客

3.cd daydream

4.hugo server -t pure –buildDrafts

5.hugo –theme=pure –baseUrl="https://loardforwin.github.io” –buildDrafts

6.cd public

7.git add .

8.git commit -m “这是我的博客”

9.cd..

10.git remote add origin https://github.com/loardforwin/loardforwin.github.io.git

11.git push -u origin master

2.博客的更新

1.hugo –theme=pure –baseUrl="https://loardforwin.github.io” –buildDrafts

2.cd public

3.git add .

4.git commit -m “这是我的博客”

5.git push -u origin master

3.创建博客的md文件

1.hugo new xxx.md
